# 成语接龙（谐音可选 + 传统胜负 + 可选「马」模式）

## When to use

当用户要求进行 **成语接龙**，或本 skill 所描述的 **谐音接龙 / 含「马」胜负** 时启用。若用户 **未** 说明启用「马」模式，则 **不** 套用「先说出含马成语者胜」的规则（见 **§B**），仅使用 **§A** 传统胜负。

开局时应用 **一句话** 复述本局约定：**是否允许谐音**、**是否启用「马」模式**、是否禁止重复成语；若用户未说明，默认：**允许谐音**、**不启用「马」模式**、**禁止重复**。

---

## 接龙规则

### 1. 基本回合

- 双方轮流，每次只出 **一个** **规范成语**（见 **§真实性**）。
- **下一手**须以 **上一手成语的最后一个字** 为起点：
  - **同字接龙**：下一成语 **首字** 与上一成语 **末字** 为 **同一汉字**。
  - **谐音接龙（本 skill 允许）**：下一成语 **首字的标准普通话读音**（拼音，**不含声调**）与上一成语 **末字** 的读音 **相同**；**声调可以不同**；**字形可以不同**。
- 每一手须 **简短标明** 接法：**「同字：」** 或 **「谐音：」** + 成语全文，以免争议。
- 本局内 **不得重复使用** 已出现过的同一成语，除非开局另有约定。

### 2. 真实性

- 只使用 **辞书、教材或通用语料可查证** 的成语；**禁止臆造、拼凑、似是而非的“成语”**。
- 若对 **是否成语** 或 **谐音是否成立** 无把握：**不要猜测**；说明不确定，请对方换一手、举证或共同查核。

---

## 胜负与终局（必须执行）

### A. 传统接龙（默认始终适用）

1. **接不下去**  
   轮到一方时，经合理尝试后 **无法** 给出符合本局规则的下一手成语，或双方认可 **该接法下几无常见可接成语**，则 **该方判负**，**对方获胜**。  
   **AI 必须一句宣告胜负**（谁因接不下去而负），**不得**用「还想玩吗」等含糊话代替终局。

2. **非成语或违规**  
   若一方所出 **不是** 公认的规范成语，或 **违反** 本局接龙规则（未用上一手末字、或用了谐音却未按本局允许且未标「谐音：」等），则 **该方立即判负**，**对方获胜**，本局结束。

3. **重复成语**（若本局禁止重复）  
   重复已出现过的成语的一方 **判负**，对方获胜。

### B. 「马」模式（仅当开局明确启用时）

- **启用条件**：用户明确说 **启用「马」模式** / **含马决胜** 等；否则 **不启用** 本条。
- 在 **无人因 §A2 已判负** 的前提下：棋盘上 **第一次** 出现 **合法一手** 且该成语 **正文中含汉字「马」** 时，**走出该手的一方获胜**，**本局立即结束**，不再续接。
- **若该手为人类**：AI **恭喜人类获胜** 并结束。  
- **若该手为 AI**：AI **宣布自己获胜** 并结束。

### C. 多种终局冲突时的优先级

1. 若已触发 **§A2（非成语或违规）** → **立即**按 §A2 结束，**不再**套用 §B。  
2. 否则，若 **已启用 §B** 且出现 **首次** 合法含「马」成语 → 按 §B 判胜并结束。  
3. 否则 → 按 **§A1 接不下去** 等传统规则判负。

### D. 可选赛制

若用户事先约定 **步数上限、时间限制、仅同字不接谐音** 等，**以约定为准**；终局时 **一句** 复述约定并宣告胜负。

---

## AI 每回合流程

1. 确认本局：**谐音是否允许**、**「马」模式是否启用**、是否禁止重复。  
2. 读取链上 **上一手末字** 与用户/AI 上一手是否已 **终局**（含「马」或判负）。  
3. 仅在有把握时给出一手：**「同字：」或「谐音：」** + 成语；否则请求澄清或认输（若轮到自己且接不上）。  
4. 若本手或用户上一手触发 **§A / §B / §C**，**必须** 按规则 **宣告胜负** 并停止续接。

---

## 输出风格

- 简洁：接法标签 + 成语；终局时 **多一句** 胜负宣告即可。  
- 不做冗长闲聊；**除非**用户明确想开新一局。

---

## 发送前自检

- [ ] 本手成语真实，未臆造。  
- [ ] 已标「同字：」或「谐音：」，且接法符合本局约定。  
- [ ] 若游戏应结束：已按 **§A / §B / §C** 宣告 **谁胜谁负及原因**，未用套话代替。
